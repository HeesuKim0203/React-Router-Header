# React Header

The customized header with [react-router-dom](https://www.npmjs.com/package/react-router-dom)

## Installation

Using [npm](https://www.npmjs.com/):

    $ npm install --save react-router-header

```js
// using ES6 modules
import {Routes, Route} from "react-router-header";
```
